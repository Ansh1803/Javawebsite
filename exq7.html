public class VowelException {
            public static void main(String[] args) {
                try {
                    checkVowels("sky");
                } catch (Exception e) {
                    System.out.println("Caught an exception: " + e.getMessage());
                }
            }
        
            public static void checkVowels(String str) throws Exception {
                if (!str.matches(".*[aeiou].*")) {
                    throw new Exception("String does not contain vowels.");
                }
            }
        }

Output:

Caught an exception: String does not contain vowels.
