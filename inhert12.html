class Vehicle {
            void start() {
                System.out.println("Vehicle is starting.");
            }
        }
        
        class Car extends Vehicle {
            void start() {
                super.start(); // Call the start method of Vehicle
                System.out.println("Car is starting.");
            }
        }
        
        public class SuperFunctionExample {
            public static void main(String[] args) {
                Car myCar = new Car();
                myCar.start();
            }
        }

Output:

Vehicle is starting. Car is starting.
